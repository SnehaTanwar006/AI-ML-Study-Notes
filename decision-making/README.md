## TOPSIS Overview

TOPSIS selects the alternative that is **closest to the ideal solution** and **farthest from the anti-ideal solution**.

**Steps:**
1. Normalize the decision matrix
2. Apply criteria weights
3. Find the ideal best and ideal worst values
4. Calculate Euclidean distance from ideal best/worst
5. Rank by similarity score (0 to 1 — higher is better)
